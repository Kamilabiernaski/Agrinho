# 🌱 Agrinho 2026

## Tema

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.**

## Assunto Escolhido

**Uso consciente da água na agricultura.**

## Sobre o Projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação Web.

A aplicação apresenta informações sobre a importância do uso consciente da água na agricultura, mostrando como práticas sustentáveis podem contribuir para aumentar a produção de alimentos e, ao mesmo tempo, preservar o meio ambiente.

O site também possui um quiz interativo desenvolvido em JavaScript para incentivar o aprendizado sobre o tema.

## Objetivo

Demonstrar que a utilização responsável da água é essencial para garantir uma agricultura sustentável, proteger os recursos naturais e promover um futuro melhor para as próximas gerações.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Inteligência Artificial como apoio no desenvolvimento

## Funcionalidades

* Página informativa sobre sustentabilidade.
* Explicação sobre o uso consciente da água.
* Quiz interativo em JavaScript.
* Layout responsivo para celular e computador.
